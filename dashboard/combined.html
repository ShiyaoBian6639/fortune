📖 μ 与 w_qp 是什么? 马科维茨的输入到底是什么?

μ 是 XGBoost 预测的次日相对沪深300超额收益(%)

  • 训练目标：y_{i,t} = pct_chg_{i,t+1} − pct_chg_{CSI300,t+1} (横截面去均值)
  • 预测输出：μ̂_{i,t+1} = E[y_i | 截止 t 的特征] — 即"该股次日相对大盘多涨几个百分点"
  • 不是绝对涨幅,不是相对昨日的涨跌,而是相对 CSI300 的超额收益
  • μ > 0 → 预测跑赢大盘; μ < 0 → 预测跑输大盘 (但仍可能绝对上涨)

w_qp 是长多头马科维茨 QP 求出的组合权重

对选中的 K = 10 只股票求解：

minimize  0.5·λ·wᵀΣw − μᵀw
s.t.      Σ wᵢ = 1,  wᵢ ≥ 0
  • 输入 1: μ 向量 — 上面 XGBoost 的 10 个预测值 (% 单位)
  • 输入 2: Σ 矩阵 (10×10) — 不是原始 pct_chg 序列,而是用过去 60 个交易日 (严格在预测日之前) 的 pct_chg 经 Ledoit-Wolf 收缩估计得到的协方差矩阵
  • 目标：最大化 μᵀw (预期收益) − 0.5·λ·wᵀΣw (波动惩罚), λ=1
  • 约束：全仓 (Σwᵢ=1) + 不许做空 (wᵢ≥0)

为什么 10 只股票多数 μ 为负?

因为 XGBoost 训练时目标已经横截面去均值,所以市场上多数股票预测会跑输大盘。同时涨跌停过滤大量剔除了 μ 最高的候选。本日具体情况：

  • 全市场 5,078 只股票中, 只有 66 只 (1.3%) μ > 0
  • μ 排名前 30 的股票 — 全部 在特征日 (2026-04-24) 触及涨停被过滤 (主板/创科 10/20%, ST 5%)
  • 排名前 60 才出现第一个未触停的候选：*ST摩登 (rank=60, μ=+0.22%)
  • 所以本组合的 10 只股票来自 rank 60–76, 其中只有 *ST摩登 仍有正 μ

这是"涨停后效应"：昨日预测的强势股已经在本日涨停封板, 下一交易日的可买池只剩次级候选。 QP 仍会给最大正 μ 的股票最高权重 (本日 *ST摩登 拿到 31.4% 权重), 其余按 Σ 风险分散到几个 μ ≈ 0 的股票上, 以满足 Σwᵢ=1 的全仓约束。 长期看,组合的预期超额收益 = wᵀμ ≈ 0.06%, 看似很小,但叠加 200+ 个交易日就形成了回测的高夏普。

仓位规模与回测保持完全一致：起始 100 万元，可投资金额 95% (留 5% 现金缓冲)， 扣除入场费 10bps、退出费 15bps (含印花税)、市场冲击 10bps×√(单笔/20日均额)， 单股不超过 20 日均成交额的 5% (流动性上限)，按 100 股最小手取整。

模型概览

XGBoost 回归模型 — 预测每只股票次日相对沪深300的超额收益。

训练流程：滚动窗口前向验证 (Walk-Forward)

采用 212 折非重叠滚动验证，每折结构：

12 周训练 → purge gap → 2 周验证 → 2 周测试

每个测试日的预测都来自训练截止日 严格早于 该测试日的某折模型， 确保完全无前视。每折独立训练 220 棵树 (或验证集早停)。

XGBoost 超参 (核心)


    

  







  
预测目标 (相对沪深300的超额收益)

  

    
目标变量定义：

    
y_{i,t} = pct_chg_{i,t+1} − pct_chg_{CSI300, t+1}

    
即每只股票次日相对沪深300指数的超额涨跌幅 (单位 %)。

    
为何用相对收益？

    
    
      
  • 剔除市场风险：模型不是去预测大盘涨跌 (无法预测)，而是预测哪些股票会跑赢/跑输大盘。
    • 
      
  • 样本平衡：在牛熊市切换时，绝对收益分布大幅偏移；超额收益分布始终接近 0 中心，
        让模型在不同市场状态下保持一致的优化方向。
    • 
      
  • 策略可执行：交易者关心的是相对排名 (做多看多的、回避看空的)。
    • 
    

    
预处理

    
    
      
  1. 截尾: 绝对值超过 ±11% 的极端样本被截断 (A股涨跌停板宽度)。
    2. 
      
  2. 横截面去均值: 每个交易日内对所有股票的目标值减去当日均值 (
        target_norm='demean')。这样按当日常数平移收益的特征 (例如 "今天大盘涨了") 无法预测目标，
        强制模型学习区分股票之间差异的信号。

Barra 风险归因

  

    将策略每日收益分解为 5 个 Barra 风格因子的贡献加上残差 α (选股 α)。
    因子暴露 = 持仓个股的因子 z-score 平均值；因子收益 = 当日按 z-score 排序后五分位
    多空价差 (top quintile − bottom quintile)。投资比例 = 当日 invested / NAV，
    用以校正现金占比导致的低估。
  

  

  

    

    

  

  

    
如何阅读

    
    
      
  • 左图 (因子暴露)：策略持仓的平均 z-score 在每个交易日的时序。
        正暴露 = 偏向高分位；负暴露 = 偏向低分位。例如 SIZE > 0 表示偏好大盘股。
    • 
      
  • 右图 (累计贡献分解)：把策略累计收益 (黑色) 拆成 5 个因子贡献 + α 残差 (红色)。
        如果策略的高收益主要由 α 解释，说明是真正的选股能力；如果由因子解释，
        则更多是风格 beta，可能是后视偏误或机制套利。
